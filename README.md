# ProtocoloHTTP

## Ejecución:

- **Trabajado en Windows, en las rutas(Request Thread) / ó \\ adecuar según sistema operativo en caso de ser necesario.**

- Insertar el template(Carpeta con archivos html, css, img, etc) en la carpeta de recursos ó pegar la ruta del template en Request Thread método obtener ruta en el return Paths.get.
